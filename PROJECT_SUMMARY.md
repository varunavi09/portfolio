## 🔧 Customization Guide

### Update Personal Information
1. Edit `src/data/skills.js` - Update skills
2. Edit `src/data/projects.js` - Update projects
3. Edit `src/data/certifications.js` - Update certifications
4. Edit `src/data/achievements.js` - Update achievements
5. Edit `src/components/Hero.jsx` - Update social links
6. Edit `src/components/Contact.jsx` - Update contact info

### Change Colors
- Edit `tailwind.config.js` for color definitions
- Update `src/index.css` for gradient colors

### Replace Images
- Update image URLs in component data
- Replace placeholder images with actual screenshots
- Update profile picture in Hero section
